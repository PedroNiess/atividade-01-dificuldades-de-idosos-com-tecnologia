Tutoriais Simples para Você

Siga as instruções passo a passo para aprender a usar seus apps favoritos.

Como acessar o YouTube

1. Abra seu navegador (como o Google Chrome).

2. Na barra de endereços, digite www.youtube.com e pressione Enter.

3. Agora você pode assistir seus vídeos favoritos!

Como usar o Google Chrome

1. O Google Chrome é um programa para navegar na internet.

2. Para usar, clique no ícone do Chrome no seu computador.

3. Se não tiver instalado, baixe clicando aqui: Baixar Google Chrome.

Como acessar o Gmail

1. Abra o navegador e digite mail.google.com.

2. Faça login com seu e-mail e senha.

3. Você verá suas mensagens e poderá enviar e receber e-mails.

Como usar o WhatsApp Web

1. Abra seu navegador e vá para web.whatsapp.com.

2. No seu celular, abra o WhatsApp e vá em Configurações > Aparelhos conectados.

3. Escaneie o código QR que apareceu na tela do computador.

4. Agora você pode conversar pelo computador.

Como fazer uma ligação no telefone

1. Pegue seu celular e abra o aplicativo "Telefone".

2. Digite o número que deseja ligar.

3. Aperte o botão verde para fazer a ligação.

Como usar o Microsoft Excel

1. Você pode usar o Excel no seu computador ou online.

2. Para usar online, vá para office.com.

3. Faça login com sua conta Microsoft.

4. Crie ou abra planilhas para organizar seus dados.
